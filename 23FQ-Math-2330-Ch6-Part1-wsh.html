6.2 - Part 1: Line Integrals wrt Arclength

Parametric Curves Review (See OpenStax Calculus, Volume 3: 1.1-1.2, 3.1, 3.3):

  

A curve C can be parametrized using a vector function:

r→(t) = x(t),y(t),a ≤ t ≤ b

Tangent Vector:

r→′(t) = x′(t),y′(t))

Unit Tangent Vector:

T→(t) = r→′(t) |r→′(t)|

Arclength:

L = ∫ abdx dt 2 + dy dt 2dt = ∫ ab|r→′(t)|dt = ∫ Cds

Line Integral of a Scalar Function with respect to Arclength:

∫ Cf(x,y)ds

denotes an integral whose domain of integration is a curve C from some starting point A to an ending point B.
To evaluate, we want to rewrite the integral in terms of the parameter t, using the distance element ds = dx dt 2 + dy dt 2dt:

∫ Cf(x,y)ds = ∫ abf(x(t),y(t))dx dt 2 + dy dt 2dt

Line Integrals with respect to x or y:

Given a curve C with parametrization r→(t) = x(t),y(t),a ≤ t ≤ b

∫ Cf(x,y)dx = ∫ abf(x(t),y(t))x′(t)dt

∫ Cf(x,y)dy = ∫ abf(x(t),y(t))y′(t)dt

As it turns out, these often occur together, so we develop a shorthand:

∫ CP(x,y)dx + ∫ CQ(x,y)dy = ∫ CP(x,y)dx + Q(x,y)dy

6.1 - Vector Fields:

Vector Field Deﬁnitions & Terminology:

A vector ﬁeld, F→ is a function that assigns a vector to each point in the domain:

F→(x,y) = P(x,y),Q(x,y)

P,Q are called the component functions, and are scalar functions.

Recall from section 4.6, we deﬁned the gradient ∇→f = fx,fy. The vector ﬁeld deﬁned by ∇→f is called the Gradient Field:

F→ = ∇→f.

 
A vector ﬁeld F→ is called conservative if

F→ = ∇→f.

for some scalar function f, which is called the potential function for F→.
