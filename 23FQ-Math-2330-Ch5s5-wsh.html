Spherical Coordinates

  

Variables:

  • ρ: distance from the origin ρ2 = x2 + y2 + z2, ρ ≥ 0
  • 𝜃: same angle as in polar/cylindrical coordinates, 0 ≤ 𝜃 ≤ 2π

Equations:
