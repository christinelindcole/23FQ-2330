6.2 - Part 2: Line Integrals over Vector Fields

  

A curve C can be parametrized using a vector function:

r→(t) = x(t),y(t),a ≤ t ≤ b

Tangent Vector:

r→′(t) = x′(t),y′(t))

Unit Tangent Vector:

T→(t) = r→′(t) |r→′(t)|

Work:

W = F→ ⋅D→ = |F→||D→|cos𝜃 = FD|D→|

Line Integrals over Vector Fields: Work

This type of line integral represents the work done by a force F→ to move a particle along the curve C from t = a to t = b:

W = ∫ CF→ ⋅ dr→ = ∫ abF→(r→(t)) ⋅r→′(t)dt = ∫ CF→ ⋅T→ds

If F→ = P(x,y),Q(x,y) and dr→ = dx,dy, then

∫ CF→ ⋅ dr→ = ∫ CP(x,y)dx + Q(x,y)dy = ∫ abP(x(t),y(t))x′(t) + Q(x(t),y(t))y′(t)dt

6.3 - The Fundamental Theorem of Calculus for Line Integrals (FTCFLI)

Path Dependence Terminology:

Consider a line integral over a vector ﬁeld from a starting point A to and ending point B:

∫ CF→ ⋅ dr→

If the value of the line integral changes depending on what path is taken from A to B, then we say that it is path dependent.
If the value of the line integral is the same for any path between A and B, then we say that it is path independent.  

0.1 FTCFLI:

If f is diﬀerentiable and ∇→f is continuous on a curve C parametrized as r→(t) for a ≤ t ≤ b, then

∫ C∇→f ⋅ dr→ = f(r→(b)) − f(r→(a)).

This means that if F→ = ∇→f for some potential function f then the line integral ∫ CF→ ⋅ dr→ is path independent.
