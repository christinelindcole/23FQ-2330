6.3 - The Fundamental Theorem of Calculus for Line Integrals (FTCFLI), Part 2:

FTCFLI:

If f is diﬀerentiable and ∇→f is continuous on a curve C parametrized as r→(t) for a ≤ t ≤ b, then

∫ C∇→f ⋅ dr→ = f(r→(b)) − f(r→(a)).

This means that if F→ = ∇→f for some potential function f then the line integral ∫ CF→ ⋅ dr→ is path independent.

Properties of Conservative Vector Fields:

F→ = P(x,y),Q(x,y) is conservative on a region R in the plane (with no holes) if:

A:
∮ CF→ ⋅ dr→ = 0 for any closed path C
B:
∫ CF→ ⋅ dr→ depends only on the endpoints, not on the path itself
C:
F→ = ∇→f for some potential function f
D:
The components satisfy the component test: ∂P ∂y = ∂Q ∂x

6.4 - Green’s Theorem:

Green’s Theorem:

  

For any vector ﬁeld F→ = P,Q that has continuous ﬁrst partials:

∮ CF→ ⋅ dr→ = ∮ CPdx + Qdy = ∬D∂Q ∂x −∂P ∂y dA,

where C is a simple closed curve that is piecewise smooth and positively oriented that encloses the region D.

  • simple closed curve: only intersects itself at the start/end points
  • piecewise smooth: can be broken up into pieces without corners
  • positively oriented: draw the arrows on C sot that the enclosed region D is always on the left as you go around C.

Area Formulas:

Green’s Theorem can be used to calculate the area of the region D by choosing P and Q so that ∂Q ∂x −∂P ∂y = 1.
The most popular options are shown below:

Area(D) = ∮ Cxdy = −∮ Cydx = 1 2∮ xdy − ydx.

Use whichever formula makes the problem the easiest!
