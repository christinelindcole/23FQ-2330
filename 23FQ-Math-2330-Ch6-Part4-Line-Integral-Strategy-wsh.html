Chapter 6 - Part 4: Line Integral Strategy

Types of Line Integrals:

Line Integral of a Scalar Function with respect to Arclength:

∫ Cf(x,y)ds

Interpretation: Net area under z = f(x,y) above the curve C in the xy−plane.

Evaluate by:

1.
Sketch & parametrize C
2.
Replace x = x(t),y = y(t),ds = dx dt 2 + dy dt 2dt
3.
Integrate wrt t from a to b.

Line Integrals with respect to x and/or y:

∫ CP(x,y)dx + Q(x,y)dy

Evaluate by:

1.
Sketch & parametrize C
2.
Replace x = x(t),y = y(t),dx = x′(t)dt,dy = y′(t)dt
3.
Integrate wrt t from a to b.

Line Integrals over Vector Fields:

∫ CF→ ⋅ dr→

Interpretation: Work done by a force F→ to move a particle along C from the starting point to the ending point.

Evaluate by:

1.
Sketch & parametrize C
2.
Replace with ∫ abF(t)→ ⋅r→′(t)dt

Miscellaneous Line Integral Problems

Problem 1.
∫ Cxdx + ydy x2 + y2 ,C: semi-circular arc of x2 + y2 = 4 from (2,0) to (−2,0).
Problem 2.
∫ Cx + 2yds,C: parametrized by r→(t) = 2 − 3t,4t − 1,0 ≤ t ≤ 2.
Problem 3.
∮ C x2 − ydx + y2 − xdy,C: circle of radius 5 centered at the origin, counterclockwise.
Problem 4.
∫ C1 + y 3ds,C: parametrized by r→(t) = 30cos3t,30sin3t,0 ≤ t ≤ π∕2.

Answers:
Problem 1: 0, Problem 2: 50, Problem 3: 0, Problem 4: 225
